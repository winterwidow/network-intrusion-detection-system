# network-intrusion-detection-system

## IDS
What is IDS?

IDS is a security tool that monitors network traffic or system activities to detet unauthorized access or suspicious behaviour.

Detectes threats such sa malware, DoS/DDoS attacks and brute-force attacks.

### Common methods of intrusion:
1.	Address spoofing – hiding the source of an attack by using a fake or unsecured proxy servers.
2.	Fragmentation – sending data in small pieced to slip  past detection systems.
3.	Pattern evasion – changing attack methods to avoid detection by IDS systems that look for specific patterns

### Detection Methods:
1.	Signature based detection:
This method relies on predefined patterns of known attacks. It compares incoming data to a database of attack signatures.
2.	Anomaly based detection:
This method uses machine learning or statistical models to define what normal behaviour looks like.

## Difference between ids and firewalls:
- Firewalls: used to prevent unauthorized access by restricting traffic between networks based on predefined rules.
- IDS: detects and alerts on suspicious activity after an attack has occurred.
